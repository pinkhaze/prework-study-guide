HTML

  • Enter your HTML notes here
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • A semantic HTML element describes its role and its contents on a webpage.
  • An element refers to the entire element from opening to closing tag.
  • A tag refers to the contents inside the angle brackets.
  • A div element creates a divide on a webpage.
  • A span element creates an inline division on a webpage.

CSS

  • Inline CSS styles an element directly in an HTML file.
  • Internal style sheet contains rules for a webpage in the head section of an HTML file.
  • External CSS style sheet contains rules for a webpage in a separate file.
  • A selector defines the element or attributes to which declarations apply.
  • Declarations contain the CSS property we want to apply and the value of said property.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switch to the main branch
  • git pull origin main: pulls remote main branch into local main branch
  • git add -A: add changed files in working directory to staging area of Git
  • git commit -m "enter your message": commit changes to repository with a message

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • The var keyword declares a variable.
  • A string is contained within quotation marks.
  • The console.log method, when provided an argument, outputs a message to the web console.
  • The script.js file is linked within the body element of an HTML file.
  • Control flow is the order in which a computer executes code in a script.
  • The scrict equality operator (===) checks if two values are equal.
  • An array is a single variable used to hold a group of data.
  • An index identifies the location of a data item in an array.
  • A function is a set of instructions that tells a computer how to perform a certain task.
  • The function keyword creates a new function.
