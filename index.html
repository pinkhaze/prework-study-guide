HTML

  • Enter your HTML notes here
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • A semantic HTML element describes its role and its contents on a webpage.
  • An element refers to the entire element from opening to closing tag.
  • A tag refers to the contents inside the angle brackets.
  • A div element creates a divide on a webpage.
  • A span element creates an inline division on a webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The style.css file is linked within the head element of an HTML file.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls remote main branch into local main branch

JavaScript

  • A variable is a named container for storing data in code.
  • The var keyword declares a variable.
  • A string is contained within quotation marks.
  • The console.log method, when provided an argument, outputs a message to the web console.
  • The script.js file is linked within the body element of an HTML file.
  • Control flow is the order in which a computer executes the code in a file.
  • The scrict equality operator (===) checks if two values are equal.
  • An array is a single variable used to hold a group of data.
  • An index identifies the location of a data item in an array.
